# Шаблон отчёта по оценке LLM

## 1. Назначение LLM-сценария

Кратко описать, где используется LLM и какую задачу помогает решать.

## 2. Набор тестовых кейсов

- ID кейса
- Тип запроса
- Ожидаемые факты
- Ответ кандидата

## 3. Метрики оценки

- Релевантность
- Фактологичность
- Полнота
- Риск галлюцинации

## 4. Вывод

Ответы с высоким риском должны проходить human review. Для production-подхода нужен golden dataset и регламент регулярной проверки.
